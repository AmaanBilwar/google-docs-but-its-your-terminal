# google-docs-but-its-your-terminal
edit docs with multiple people inside your terminal

## Setup 
1. clone the project and cd into it: 
```zsh
git clone https://github.com/AmaanBilwar/google-docs-but-its-your-terminal.git && cd google-docs-but-its-your-terminal
```
2. run in your preferred terminal of choice
```zsh
cargo run --release
```
3. Boom
